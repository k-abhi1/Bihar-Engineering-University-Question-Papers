# BeuAdhyay – Engineering Study Portal

BeuAdhyay is a comprehensive web platform designed for students of Bihar Engineering University (BEU). It provides easy access to previous year question papers, syllabus, scholarships, holiday calendars, and an AI‑powered assistant – all in one place.

## 🌟 Features

- **Dashboard** – Browse question papers by branch and semester with working view/download buttons.
- **Syllabus** – Access detailed syllabus for all branches and semesters.
- **University Notifications** – Direct link to official BEU notifications.
- **Scholarships** – Information on various scholarships (PMS, NSP, NPTEL, AICTE internships, etc.).
- **Holidays** – Academic calendar and list of holidays.
- **Results** – Quick link to BEU result portal.
- **AI Assistant** – Chatbot that answers common academic queries (e.g., syllabus, exams, scholarships).
- **Mobile‑friendly** – Responsive design with a hamburger menu for smaller screens.
- **Dark mode support** – Automatically adapts to user’s system preference.

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Custom properties (variables), Flexbox, Grid, and comprehensive media queries
- **JavaScript (ES6)** – Dynamic content loading, event handling, and simple AI logic
- **Font Awesome** – Icons
- **Google Fonts** – Poppins & Inter fonts
